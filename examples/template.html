Try out some of these keyboard shortcuts!
  • Use TAB to focus the button
  • Use ENTER to open the menu
  • Use the ↑ and ↓ ARROW KEYS to navigate between items
  • Use ENTER again to select a highlighted item
  • Use TAB again to select the "Create" button and use ENTER to create an item
  • Use ESCAPE to close the menu
